## 垃圾字体鉴定器（两组很像的符号）：

1. 丨｜|Il1¡i<br>

```
丨 中日韩象形文字 中日韩统一表意文字
｜ 全形竖线 半角及全角形式
| 竖线 基本拉丁字母
I 拉丁文大写字母i 基本拉丁字母
l 拉丁文小写字母L 基本拉丁字母
1 数字1 基本拉丁字母
i 拉丁文小写字母i 基本拉丁字母
¡ 竖翻叹号 拉丁文补充1
```

2. 0oO〇<br>

```
0 数字0 基本拉丁字母
o 拉丁文小写字母o 基本拉丁字母
O 拉丁文大写字母O 基本拉丁字母
〇 象形字数字0 中日韩符号和标点
```
